Powerful Features for Success

Everything you need to prepare for the LET examination in one place

For Students

Offline Mode

Full access to quizzes without internet. Uses local database (SQLite / IndexedDB) so you can study anytime, anywhere.

Installable App

Install as PWA or APK (via Capacitor) on your device. Access from home screen with a native app-like experience.

Review & Quiz System

Categorized modules (GenEd, ProfEd, Majorship), multiple choice questions randomized per attempt, and timed exam simulation with auto-submit.

Progress Tracking

Scores per category, quiz history with attempt logs, performance insights identifying weak/strong topics with simple recommendations.

Engagement Features

Offline notifications reminding "Time to review!", gamification with badges (e.g., "GenEd Master"), and daily streaks to keep you motivated.

For Teachers / Admin

Web-Based Admin Panel

Accessible via browser as a separate system. Manage everything from one centralized online dashboard.

Content Management System

Create/edit MCQs with explanations, organize subjects/topics with category management, bulk upload via CSV/Excel, and add images/diagrams.

Sync & Deployment

Push updates to student apps. Students download updates when online through a pull-based sync strategy.

Monitoring

Student performance dashboard showing average scores per category, user insights with active users and quiz attempts.

User Management

Teacher accounts with role-based access (Admin, Teacher) for secure and organized content management.
